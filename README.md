# Vichar
A QA Webapp where user can post, delete, like and comment on queries

---

# TechStack

- React
- Node
- Express
- Mongodb

---

# Project Structure

```
$ROOT
├── client
│   ├── src
│   │   # react component files
│   └── public
│       # Static files 
└── server
    # Node server files
```
# Setup

- Declare PORT and CONNECTION_URL in .env
- run ```npm i``` to install packages
